# Crypto ML Service

Короткий учебный сервис для работы с ML-прогнозами по криптовалютам. Проект включает FastAPI API, простую веб-страницу личного кабинета, PostgreSQL для хранения пользователей/баланса/истории, RabbitMQ для очереди задач и ML-воркеры для обработки прогнозов.

## Что умеет сервис

- регистрация и авторизация пользователей;
- пополнение и просмотр баланса;
- список доступных ML-моделей;
- создание задач на прогноз;
- просмотр статуса прогноза и истории операций.

## Запуск через Docker

Нужны установленные Docker и Docker Compose.

```bash
docker compose up --build
```

После запуска:

- веб-интерфейс: `http://localhost:8080`
- API healthcheck: `http://localhost:8080/health`
- RabbitMQ Management UI: `http://localhost:15672`
